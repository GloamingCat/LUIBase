# LUI Base

Common code for my projects LWT and LSwing, two available libraries to be used as a base for my Lovely Tactics Hex Editor: https://github.com/GloamingCat/LTH-Editor.
* LSwing: https://github.com/GloamingCat/LSwing
* LWT: https://github.com/GloamingCat/LWT

## Installation

* This project is made with IntelliJ IDEA Community Edition 2023.3.6.
* This project needs the following libraries added to the project's dependencies:
  * GSON 2.2.2
  * ini4j
